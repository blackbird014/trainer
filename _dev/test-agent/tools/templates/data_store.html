Data Store API

API URL: {{ api_url }}

The data-store module provides a REST API for querying stored data.

Available Endpoints

  • GET {{ api_url }}/health - Health check
  • GET {{ api_url }}/query - Query stored data
  • GET {{ api_url }}/retrieve/{key} - Retrieve by key
  • GET {{ api_url }}/count - Count items
  • GET {{ api_url }}/metrics - Prometheus metrics
View API Documentation

Quick Access

Use the data-store API to browse stored data. You can also use the MongoDB shell directly:

docker exec mongodb mongosh trainer_data
